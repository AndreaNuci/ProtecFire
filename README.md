# ProtecFire

ProtecFire es un proyecto que desarrolla una aplicación web integrada con tecnología IoT para la detección y prevención de incendios.

El sistema permite monitorear en tiempo real variables como temperatura, humo y gases mediante sensores conectados a dispositivos IoT. Estos sensores envían los datos a una plataforma web, donde se procesan y analizan para identificar posibles riesgos.

La aplicación genera alertas automáticas cuando se detectan valores fuera de los parámetros normales, además de permitir la visualización de zonas críticas para facilitar la toma de decisiones y actuar oportunamente ante emergencias.

## Características principales

- Monitoreo en tiempo real.
- Integración con sensores IoT.
- Generación automática de alertas.
- Visualización de datos y zonas críticas.
- Prevención y detección temprana de incendios.

## Objetivo

Desarrollar una solución tecnológica que contribuya a la prevención de incendios mediante el uso de sensores inteligentes y una plataforma web de monitoreo accesible y eficiente.
